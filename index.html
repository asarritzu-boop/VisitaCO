        Premessa del Sorvegliante di Circoscrizione

        Cari fratelli anziani,

        con piacere vi condivido questo strumento digitale che ho predisposto per organizzare in modo ordinato e pratico la settimana di visita alla vostra congregazione. L'app «Visita del Sorvegliante di Circoscrizione» è un'applicazione web che funziona direttamente nel browser del vostro telefono o computer — senza bisogno di installare nulla — e permette di raccogliere e organizzare tutte le informazioni relative alla settimana di visita.

        Questo manuale vi guiderà passo dopo passo su come usarla. Vi chiedo di leggerlo con attenzione e di compilare le sezioni di vostra competenza con cura, in modo che io possa ricevere un quadro completo e ben organizzato prima di arrivare.

        ℹ️ Ogni anziano è responsabile della compilazione delle sezioni della propria congregazione. Al termine, dovrà esportare il backup e inviarmelo. Sarò io a raccogliere e unire i dati di tutte le congregazioni della circoscrizione.
        2

        Cos'è l'App e come aprirla

        L'app è un file HTML che vi è stato condiviso direttamente da me. Non richiede installazione: funziona direttamente nel browser.

        1
        Ricevete il file a tempo debito.
        2
        Salvate il file sul vostro dispositivo (telefono o computer), ad esempio nella cartella Download.
        3
        Toccate (o fate doppio clic) sul file: si aprirà automaticamente nel browser (Chrome, Safari, Firefox ecc.).
        4
        Per comodità, aggiungete la pagina ai Preferiti del browser oppure usate la funzione «Aggiungi alla schermata Home» del telefono per ritrovarla subito.
        ℹ️ Non è necessaria una connessione Internet per usare l'app. Tutti i dati vengono salvati direttamente nel browser del vostro dispositivo.
        3

        Primo avvio: creare la settimana di visita

        Quando aprite l'app per la prima volta, vedrete la schermata principale con l'intestazione «Visita del Sorvegliante di Circoscrizione» e un elenco vuoto. Il primo passo è creare la settimana relativa alla vostra congregazione.

        1
        Toccate il pulsante ＋ Aggiungi settimana in alto a destra.
        2
        Si aprirà un riquadro con i campi da compilare.
        3
        Inserite il nome della congregazione (es. Castelsardo).
        4
        Inserite il nome del Sorvegliante di Circoscrizione (il mio nome).
        5
        Se la mia moglie parteciperà agli appuntamenti, inserite anche il suo nome nel campo apposito.
        6
        Selezionate la data di inizio e la data di fine della settimana di visita usando il calendario che appare.
        7
        Toccate Conferma per salvare. La settimana apparirà nella lista principale.
        8
        Toccate sulla settimana appena creata per aprirla e iniziare a compilare le sezioni.
        ℹ️ Se commettete un errore, potete modificare la settimana toccando l'icona ✏️ accanto al suo nome, oppure eliminarla con l'icona 🗑️ e ricrearla.
        4

        📝 4.7 — Punti OdG

        Questa sezione permette di preparare i punti dell'ordine del giorno per l'adunanza degli anziani. Ogni punto dispone di un titolo, un'area di testo ricca con formattazione, e — novità importante — la possibilità di allegare uno o più file direttamente alle Fonti di ciascun argomento, visualizzabili in anteprima anche senza connessione Internet.

        Come creare e compilare un punto OdG

        1
        Nella parte superiore della pagina, inserite il nome della congregazione nel campo in alto.
        2
        Toccate + aggiungi punto per creare un nuovo argomento.
        3
        Nel campo Argomento, scrivete il titolo del punto (es. Revisione territorio, Attività di predicazione).
        4
        Nel campo Fonti, potete scrivere i riferimenti testuali (es. Lettera della filiale del 12/04) oppure allegare uno o più file — vedi la sezione qui sotto.
        5
        Nel campo Trattato da, indicate il nome del fratello incaricato di esporre il punto.
        6
        Toccate + sottotitolo per aggiungere sotto-argomenti con aree di testo formattabili (grassetto, corsivo, sottolineato, colori, link, tabelle).
        7
        Con i pulsanti ▲ e ▼ potete riordinare i punti. Con il pulsante ✕ potete eliminarne uno.
        8
        Nel campo NOTE AGGIUNTIVE in fondo, potete inserire osservazioni finali fuori dai punti standard.
        9
        Toccate 📄 Genera PDF per produrre il documento OdG pronto per la stampa o la condivisione.

        📎 Come allegare file alle Fonti di un punto OdG

        Accanto al campo di testo Fonti di ogni argomento, trovate il pulsante 📎 Allega. Questo vi permette di collegare documenti, immagini o altri file direttamente a quel punto dell'ordine del giorno. I file vengono convertiti e salvati all'interno dell'app, quindi sono sempre disponibili anche senza Internet.

        Anteprima del campo Fonti con allegati
        📎 Allega
        📄 circolare-aprile.pdf ✕
        🖼️ grafico-predicazione.png ✕
        📝 relazione-SM.docx ✕

        Procedura passo-passo per allegare un file

        1
        Individuate il punto OdG a cui volete allegare un documento e cercate il campo Fonti.
        2
        Toccate il pulsante 📎 Allega alla destra del campo Fonti. Si aprirà il selezionatore file del vostro dispositivo.
        3
        Navigate nella memoria interna del dispositivo e selezionate uno o più file. Potete allegare più file contemporaneamente.
        4
        Attendete un istante: il file viene letto e convertito. Comparirà una piccola pillola/chip con il nome del file sotto il campo Fonti.
        5
        Per eliminare un allegato, toccate la ✕ rossa sulla pillola. Vi verrà chiesta conferma prima di procedere.
        6
        Per visualizzare un allegato, toccate la pillola (non sulla ✕ ma sul nome del file). Si aprirà un pannello di anteprima a schermo intero.
        ⚠️ Nota sul selezionatore file su Android: se il file non viene letto correttamente, provate a navigare nella cartella «Memoria interna» invece di usare la schermata «Recenti». Alcuni dispositivi Android non permettono la lettura diretta dai Recenti.

        3

        4

        📄 Toccate 📄 Genera PDF per produrre il documento OdG completo, pronto per la stampa. I file allegati non vengono incorporati nel PDF, ma rimangono accessibili nell'app.

        Come vengono salvati i dati

        L'app salva i dati automaticamente ad ogni modifica, direttamente nella memoria del browser del dispositivo in uso. Non è necessario premere alcun pulsante di salvataggio manuale.

        Cosa significa concretamente:

        • Se chiudete il browser e lo riaprite, i dati saranno ancora presenti.
        • Se spegnete il dispositivo e lo riaccendete, i dati saranno ancora presenti.
        • I dati rimangono sul dispositivo dove li avete inseriti: non si sincronizzano automaticamente con altri dispositivi.
        • Se cancellate la cache o i dati del browser, i dati potrebbero andare persi.
        ⚠️ IMPORTANTE: i dati sono salvati solo su quel dispositivo specifico. Non usate browser diversi o dispositivi diversi per compilare gli stessi dati. Esportate sempre il backup prima di inviarmelo.
        6

        Esportare il Backup

        Una volta compilate tutte le sezioni di vostra competenza, dovete esportare il backup della settimana e inviarmelo. Il backup è un file .json che contiene tutti i dati inseriti, inclusi gli allegati ai punti OdG.

        1
        Dalla schermata principale, toccate sulla settimana che volete esportare per aprirla.
        2
        In basso a destra, toccate il pulsante rotondo ＋ (FAB) per aprire il menu delle azioni.
        3
        Nel menu che appare, toccate «Esporta Backup» oppure l'icona 📤.
        4
        Il browser vi chiederà dove salvare il file: salvatelo sul dispositivo (es. nella cartella Download).
        5
        Il file avrà un nome del tipo: backup_[Sorvegliante]_[data-inizio].json (es. backup_Mario_Rossi_2026-05-11.json).
        6
        Inviatemi questo file tramite WhatsApp, email o altro mezzo concordato.
        ℹ️ Il backup include tutti i dati: adunanze, servizio, comitive, ospitalità, visite pastorali, punti OdG con i relativi allegati, e archivio. È sufficiente inviarne uno solo per congregazione.

        Quando inviarmi il backup? Vi chiedo di farlo almeno 2 giorni prima dell'inizio della visita.

        7

        Il CO raccoglie i dati

        Questa sezione illustra il processo di raccolta dati da parte mia come Sorvegliante di Circoscrizione, in modo che capiate l'intero flusso.

        1
        Ricevo i file .json di backup da ciascuna congregazione della circoscrizione.
        2
        Apro la mia copia dell'app nel browser del mio dispositivo.
        3
        Dal menu FAB (＋), tocco «Importa Backup».
        4
        Seleziono il file .json ricevuto da una congregazione.
        5
        L'app importa automaticamente la settimana, aggiungendola a quelle già presenti. Non sovrascrive i dati esistenti.
        6
        Ripeto per ciascuna congregazione della circoscrizione.
        7
        Ho così tutte le settimane in un unico posto, con allegati OdG e tutti i dati, pronti per visualizzarli e generare i PDF definitivi.
        ℹ️ L'importazione è intelligente: se una settimana con lo stesso nome e data è già presente, l'app non la duplica. Se è nuova, viene aggiunta senza problemi.

        Domande frequenti

        ❓Ho cancellato accidentalmente la cache del browser. Ho perso i dati?
        Se non avevi ancora esportato il backup, purtroppo i dati potrebbero essere andati persi. Per questo raccomandiamo di esportare il backup appena finite di compilare ogni sezione importante e di conservare il file .json in un posto sicuro (es. nella chat WhatsApp con me o in una cartella dedicata).
        ❓Posso usare l'app senza avere Internet?
        Sì. Una volta che il file è aperto nel browser, l'app funziona completamente offline. Anche le anteprime degli allegati ai punti OdG funzionano senza connessione, perché i dati sono già salvati nel dispositivo.
        ❓Perché il file allegato non si carica quando lo seleziono?
        Su alcuni dispositivi Android, i file selezionati dalla schermata «Recenti» non vengono letti correttamente. Provate ad aprire il selezionatore file e a navigare manualmente nella cartella «Memoria interna» o «Download» per trovare il file.
        ❓Gli allegati ai punti OdG vengono inclusi nel backup?
        Sì. I file allegati vengono inclusi automaticamente nel file di backup .json. Quando mi inviate il backup, gli allegati sono già dentro. Non è necessario inviare i file separatamente.
        ❓Posso usare l'app su tablet o computer oltre che sul telefono?
        Assolutamente sì. L'interfaccia si adatta a schermi di diverse dimensioni. Su un computer potreste trovare più comoda la compilazione dei campi di testo più lunghi.
        ❓Dove trovo il pulsante per esportare il backup?
        Dalla schermata principale, aprite la settimana desiderata. Poi toccate il pulsante rotondo (＋) in basso a destra: si apre un menu con l'opzione «Esporta Backup». Il file verrà scaricato nella cartella Download.
        ❓Il file di backup .json è illeggibile. È normale?
        Sì, è normale. Il formato .json è tecnico, pensato per le macchine. Non è necessario aprirlo o modificarlo: è sufficiente inviarmelo così com'è.
